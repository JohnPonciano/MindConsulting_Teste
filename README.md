## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/JohnPonciano/projeto_blogpost.git
```

Entre no diretório do projeto

```bash
  cd back
```

Instale as dependências

```bash
  npm install
```

Inicie o servidor

```bash
  npm run start
```
